# NammaShaaleInventoryapp

NammaShaaleInventoryapp is an Android-based school inventory management application developed to help schools manage and track assets efficiently. The application allows users to monitor equipment status, maintain inventory records, and simplify asset management processes in educational institutions.

## Features

* Add and manage school assets
* Track equipment condition and availability
* Dashboard for inventory overview
* User-friendly Material 3 UI design
* Firebase integration for data storage
* Responsive Android application

## Technologies Used

* Kotlin
* Android Studio
* Jetpack Compose
* Firebase
* Material 3 Design

## Project Purpose

This project was developed as an academic and practical learning project to improve inventory management in schools through a simple and modern mobile application.
